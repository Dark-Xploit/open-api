## WEB 0.1

> Simple web with router plugin system

### Installation & Run

```
$ yarn
$ node .
```

or want to use pm2

```
$ yarn
$ npm i -g pm2
$ pm2 start index.js && pm2 save && pm2 logs
```

### Example API
